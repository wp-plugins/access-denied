== Description ==


Access Denied will help you enable block visitors by ip address. you can block unlimited visitors by ip address.

### Access Denied by http://paratheme.com

* Fully safe.
* Easy to use.

* Easy enable or disable ip blocking.
* Block login form submission.
* Super Lightweight plugin.



== Installation ==

1. Install as regular WordPress plugin.<br />
2. Go your plugin setting via WordPress Dashboard and find "<strong>Access Denied</strong>" activate it.<br />





After activate plugin you will see "Access Denied" menu at left side on WordPress dashboard click "Access Denied" and use the options field<br />

<br />
<strong>How to add an ip address</strong><br />
you will see an textarea "IP List" on Access Denied Settings page, write ip address separated by commas, no blank space, no new line allowed.

after fill "IP List" please check mark "Enabled IP Block ?" to confirm ip blocking, you can disable anytime this feature by uncheck this checkbox, no need to remove ip address from list.

in the same way you can block login form by "Enable Login Form Block ?", login form will block wp-login.php login form submission for IP List you entered.

== Frequently Asked Questions ==

= My own IP not blocking ? =
Please confirm you entered atleast two ip address on "IP List", first one is dummy, and from second ip address is working ip address, that's mean you must have entered a dummy ip address at first and then other ip address you want block. you can test by your own ip address.
